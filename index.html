Pedra, Papel ou Tesoura!

Jogador

I. Artificial

O jogo Pedra, Papel e Tesoura, foi desenvolvido como forma de estudo para a linguagem de Programação Javascript Pura.

Neste game você apenas seleciona qual das 3 opções do lado esquerdo deseja jogar: pedra, papel e tesoura. No mesmo instante a pequena IA jogará a opção dela, e aparecerá no topo o vencedor do game.

Jogo desenvolvido por Daniel Mota
